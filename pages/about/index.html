About This Blog

這是我部落格的第四個版本：

  • 第一個版本是在家裡的 PC 用 MovableType 架的，當然已經不存在了。
  • 第二個版本則是架在 wordpress.com 上的
  • 第三個版本則是用 Octopress 跑在 heroku.com 上的，然後又搬到 github.io 上。
  • 目前的版本則是用 Hugo 產生的，仍然放在 github.io 上。

由此可以看到 web 技術一直在演進，程式語言從 Perl 到 PHP 再到 Ruby 及 Go，內容從以前的硬刻 HTML/JavaScript/CSS 到今天流行的 Markdown/jQuery/Bootstrap，佈署方式從自家 PC 到自己架 server 然後又變直接放在雲端。

與此同時，軟體開發的技術也是一直在進步變化中，我本身從事軟體設計的平台也從 PC 轉到 feature phone 再轉到 smartphone 及 cloud computing，而這個部落格就是為了記載我的學習旅程而存在。
